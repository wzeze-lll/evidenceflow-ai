## 截图方法

推荐使用以下方式截图：

1. **Chrome DevTools 截图**：`Cmd+Shift+P` → "Capture full size screenshot"
2. **macOS 截图**：`Cmd+Shift+4` 区域截图
3. **自动化截图**（可选）：使用 Playwright 或 Puppeteer

## 截图参数

- 分辨率：1920×1080
- 主题：推荐分别截取 Light 和 Dark 模式
- 确保使用 Demo 工作区数据
- 无开发者工具、无错误提示
